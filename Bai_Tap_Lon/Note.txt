This is the program calculating the infix
In mathematics, an infix operator is a function that applies a calculation to arguments on either side of the function operand

Operators for program: + - * / ^
functions for program: sin(), cos(), exp(), log(), !, sqrt()

Take one minute to read our notes before using program
Note:
1. The value that you enter for the sin() and cos() functions is radian.
2. Use parentheses with negative numbers.
Ex: (-5), sin(-90),...
3. The program does not support calculations on the field of real numbers.
